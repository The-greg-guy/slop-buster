§ 01 — THE TWO ANCHOR PATTERNS

Named verbatim in Anthropic's Opus 4.8 prompt guide.

The two patterns the user identified — "random words in italics" and "bone color" — are not invented tells. They appear together in a single paragraph of Anthropic's official documentation as the named house style of Claude Opus 4.8.

Warm cream/off-white background paired with warm-charcoal text and a terracotta/sage/dust-rose accent. The 2026 replacement for VibeCode Purple. Identical across vet clinic, fintech, and crypto.

Fix: Pick a different accent. Mercury uses indigo #5266EB. Substack uses orange #FF6719. 925 Studios uses deep teal #0D7377.
PATTERN #2

Random words in italics

Build your next idea with confidence.
Top CTA: "Ready to automate everything?"

<em> tags mid-sentence in hero copy, feature titles, CTA labels. Verbal fry in typographic form. When every sentence has an italic accent, none of them do. Impeccable quantifies this at 28-34% slop rate on editorial/food briefs.

Fix: Rewrite in roman. If emphasis is needed, use weight contrast (800 vs 400) or size contrast (3x). Cut italic from feature copy entirely.
SOURCE — platform.claude.com/docs/en/build-with-claude/prompt-engineering/prompting-claude-opus-4-8
"Claude Opus 4.8 has strong design instincts, with a consistent default house style: warm cream/off-white backgrounds (~#F4F1EA), serif display type (Georgia, Fraunces, Playfair), italic word-accents, and a terracotta/amber accent. This reads well for editorial, hospitality, and portfolio briefs, but will feel off for dashboards, dev tools, fintech, healthcare, or enterprise apps."

§ 04 — MODEL-SPECIFIC FINGERPRINTS

Different models need different overrides.

When you identify slop, name the model that likely produced it. Each model has its own default palette, typography, and structural reflexes. The Ultimate skill package ships this table verbatim.

ANTHROPIC'S OWN ANTI-SLOP FRAGMENT — drop into your CLAUDE.md or DESIGN.md
NEVER use generic AI-generated aesthetics like overused font families
(Inter, Roboto, Arial, system fonts), cliched color schemes (particularly
purple gradients on white or dark backgrounds), predictable layouts and
component patterns, and cookie-cutter design that lacks context-specific
character.
From the October 2025 Frontend Aesthetics Cookbook. This fragment alone breaks about 60% of Claude Design defaults. The remaining 40% — the bone palette and italic word-accents — requires the explicit palette + typography constraint.
